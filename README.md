После коррекции характеристик на вязкость **КПД η изменён** — и мы пересчитываем мощность каждой секции.  
Двигатель подбирается с запасом **≥15%** от реальной потребляемой мощности, а не от табличной.

### 4. Температурная совместимость
Гидрозащита и двигатель проверяются по максимальной рабочей температуре.  
Если пластовая жидкость горячее 120°C — будут предложены только высокотемпературные исполнения.

### 5. Конструктивные ограничения
- Диаметр ПЭД и гидрозащиты не должен превышать внутренний диаметр обсадной колонны
- Максимально допустимая мощность насоса (прямой пуск / плавный пуск) не должна быть превышена
- Число секций насоса ограничено 20 — исключены нереализуемые сборки

## Как пользоваться

1. Укажите дебит, требуемый напор (или включите авто-расчёт по динамическому уровню).
2. Выберите диаметр обсадной колонны.
3. При необходимости скорректируйте плотность, вязкость, температуру.
4. Нажмите **«Подобрать оборудование»**.
5. Анализируйте отсортированный список вариантов:
   - интерактивные графики напора, мощности и КПД (точку можно двигать мышью)
   - цветовые подсказки: 🟢 норма / 🟠 на границе / 🔴 риск
   - экспорт графиков в PNG для отчёта
